# 迁移网站

迁移网站只需要按照这篇文章来就行：[hexo：更换电脑，如何继续写博客](https://blog.csdn.net/eternity1118_/article/details/71194395?ref=myread)

<!-- more -->

首先第一步，将电脑中的这些文件拷贝出来：

```bash
_config.yml
package.json
scaffolds/
source/
themes/
```

在目录中安装 hexo：

```bash
npm install -g hexo
```

接下来在目录中执行以下命令来安装插件：

```bash
npm install
npm install hexo-deployer-git --save
npm install hexo-generator-feed --save
npm install hexo-generator-sitemap --save
```

接下来就可以愉快地玩耍了。
